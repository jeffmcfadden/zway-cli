# Zway::Cli

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'zway-cli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zway-cli

## Usage

    $ zway login

Then edit `~/.zway_config`

    $ zway ZWayVDev_zway_16-0-38 level
    $ zway ZWayVDev_zway_16-0-38 on
